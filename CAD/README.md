build it from cad <br/>
some notes: <br/>
* don't overtighten z joint screws
* you need to replace z joints by sliding them all the way to the maximum z they can reach and reuse those screws
* the z belt clip things require you to snip off the locating tabs inside (you will understand once you get there)
* assembly of AB joints are convoluted, you need to put in the 2 m3x40 BEFORE attaching any motor screws
* Idlers are BFI (beefy front idlers) style so make sure when tensioning to not misalign them.
* XY joints require you to reuse the screws for mounting to carriage and also mounting the x beam
* any screw that looks like it threads into a 5mm dia hole needs a heat set (i was too lazy to add them to the cad)
* also future cad models for different versions (awd, 9mm etc) will NOT be including screws and nuts in the cad cus it took way too long
